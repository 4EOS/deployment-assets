How to Update Your Outlook Signature

Download the Template

This email has an SignatureTemplate.html file attached

Click on it to Download the attachment to your computer.

Open and Copy the Template

  1. Open the downloaded SignatureTemplate.html file (it will open in your web browser)

  2. Select all the content (Ctrl+A or click and drag)

  3. Copy it (Ctrl+C or right-click → Copy)

Open Outlook Signature Settings

  1. Go to outlook.office.com

  2. Click the Settings gear icon (top right)

  3. Click Account → Signatures → New Signature (if you already have one)

Create New Signature

  1. Name it something, e.g. "Medical Ventures"

  2. Click in the signature editor box

  3. Paste the signature (Ctrl+V or right-click → Paste)

  4. Update your Name & Email

    Note: Type your name in before deleting the placeholder name as shown below, otherwise it will lose the text formatting.

  5. Click Save

Set as Default

  1. If needed, under "Select default signatures":

    • For New Messages: Select Medical Ventures
    • For Replies/Forwards: Select Medical Ventures

  2. Click the blue Save button at the bottom.

And you're done! Your signature will now appear on all emails.
